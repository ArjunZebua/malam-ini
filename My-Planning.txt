### **1. Pendidikan & Pengembangan Diri**
- **Lulus dengan nilai terbaik di bidang Ilmu Komputer**  
  Arjun sedang mengejar gelar Ilmu Komputer, jadi fokus pada menyelesaikan tugas-tugas akademik dengan baik, termasuk proyek akhir atau skripsi.  

- **Menguasai Bahasa Pemrograman Baru**  
  Pilih satu atau dua bahasa pemrograman untuk dipelajari secara mendalam, seperti **Go** untuk performa backend atau **Rust** untuk keamanan dan efisiensi.  

- **Belajar Teknologi Baru**  
  - Kubernetes atau Docker untuk DevOps.  
  - Framework baru seperti Next.js atau SvelteKit untuk web development.  
  - Fokus pada AI/ML jika ingin eksplorasi bidang baru.
  - Belajar bahasa pemrograman C sharp
  - Belajar Laravel
  - Belajar Java

- **Ikut Kursus Online atau Bootcamp**  
  Daftar kursus di platform seperti Coursera atau Udemy untuk mendalami topik seperti **React Advanced**, **Machine Learning**, atau **Pengembangan API dengan Node.js**.

### **2. Proyek & Pengalaman Kerja**
- **Membangun Portofolio Pribadi**  
  Buat portofolio web untuk menampilkan proyek, keterampilan, dan pengalaman. Sertakan:
  - **Sepokat Store**: Refactor dan tambahkan fitur menarik.  
  - **Dunia Brended**: Tingkatkan fitur grid dan pencarian.  

- **Kontribusi Open Source**  
  Ikut serta dalam proyek open source untuk pengalaman nyata, seperti kontribusi di GitHub.  

- **Magang atau Pekerjaan Paruh Waktu**  
  Cari magang di perusahaan teknologi untuk memperkaya pengalaman praktis.  

- **Proyek Baru untuk Portofolio**  
  - Aplikasi berbasis **AI Chatbot**.  
  - Game interaktif sederhana dengan React.  
  - Sistem E-commerce (misalnya, tas atau pakaian) dengan database MySQL.

  - Ikut **hackathon** dan kompetisi untuk pengalaman dan hadiah.  

- **Melamar Pekerjaan**  
  Targetkan perusahaan teknologi terkemuka atau startup lokal. Fokus pada posisi seperti **Front-End Developer**, **Back-End Developer**, atau **Full-Stack Developer**.  

- **Ikut Komunitas Teknologi**  
  Bergabung dengan komunitas teknologi seperti Discord developer groups, Meetup, atau komunitas React lokal.
